🚀 사용 방법

빌드 및 실행

# 빌드
make clean && make
# 실행
make run
# 또는 직접 실행
./ai_detection_system [config_file.json]

# Doxygen 설치 (필요시)
make install-docs-deps
# 문서 생성
make docs
# 생성된 문서는 docs/html/index.html 에서 확인
firefox docs/html/index.html
